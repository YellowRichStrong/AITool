Frequently Asked Questions

How to submit new AI tools?

Please send email to tankeapp@gmail.com, include: tool name, official website link, function description, tool category, whether it's free, etc. We'll add it to the website after review.

What if tool information is inaccurate?

If you find inaccurate or outdated information about a tool (such as price, function description, etc.), please let us know via email, and we'll update it promptly.

Can tools be removed?

If you're the owner of a tool and want it removed from our list, please send us an email explaining the reason. We'll process it within 24 hours of receiving the request.

What languages does the website support?

Currently we support 10 languages: English, Simplified Chinese, Traditional Chinese, Japanese, Korean, Spanish, French, German, Portuguese, and Russian. You can switch via the language selector in the top right corner.

How to get tool tutorials?

We're gradually adding tool tutorials and review articles. You can also request tutorials for specific tools via email, and we'll prioritize creating them.

What business cooperation options are available?

We accept tool listing recommendations, advertising, content cooperation, and more. Please contact us via email for specific cooperation plans.

Email Template Suggestions

To help us process your email faster, please include the following information:

  • Email Subject: Clearly state the type of issue (e.g., "Bug Report", "Tool Submission", etc.)
  • Detailed Description: Specifically describe your question or request
  • Screenshots: If it's a technical issue, screenshots would be very helpful
  • Contact Information: If you need a phone call, please leave your contact details
  • Urgency: Indicate the priority of the issue
